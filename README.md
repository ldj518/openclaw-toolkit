# OpenClaw Toolkit (One-Command Install)

中文见 `README-ZH.md`

## Quick install

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/<user>/<repo>/main/bootstrap-toolkit.sh)" -- <user>/<repo> main
```

## After install

```bash
bash /root/.openclaw/workspace/openclaw-toolkit/ops/onekit-menu-zh.sh
```
